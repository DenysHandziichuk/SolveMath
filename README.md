## 🧭 Simple Workflow
1. Open the SolveMath app and create an account 📊
2. Select your math topic of choice and start a new lesson 📚
3. Work through interactive exercises and problems 🤔
4. Use the AI-powered math problem solver to get help when you need it 🤖
5. Track your progress and earn rewards and badges 🎉
6. Share your progress with friends and teachers on social media 📱

## 🎯 Purpose
The purpose of SolveMath is to make math learning fun, engaging, and accessible to everyone 🌎. By providing a comprehensive and interactive math platform, we aim to improve math skills and confidence, and to make math a enjoyable subject for all 🎉.

---
## 🧩 Installation & Usage
To get started with SolveMath, simply follow these steps:
```bash
git clone https://github.com/DenysHandziichuk/SolveMath
cd SolveMath
npm install
npm run dev
```
Then, open your web browser and navigate to http://localhost:3000 to start using SolveMath 🚀.
